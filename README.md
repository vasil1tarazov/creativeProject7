## Кратко о проекте

Программа позволяет удалённо отслеживать температуру, влажность, уровень CO₂ и присутствие людей в учебной аудитории. На основе этих данных система автоматически управляет вентиляцией и кондиционером, а также прогнозирует уровень CO₂ на 5 минут вперёд. Результаты отображаются в виде графиков.

**Основные возможности:**
- Мониторинг в реальном времени (графики температуры и CO₂)
- Прогноз CO₂ (линейная регрессия)
- Автоматическое управление: вентиляция при CO₂ > 900 ppm, кондиционер при t > 25°C
- Ручное управление устройствами через интерфейс
- Симуляция проверки температуры человека при входе (порог 37.2°C)

2. Нажмите зелёную кнопку **«Code»**.

4. Распакуйте архив в любую папку на компьютере, не содержащую кириллицу.


## Использование программы

Требования: Windows 10/11, установленный Qt 6.11.0 (MinGW 64-bit) и CMake (идут в составе Qt Creator).

1. Откройте Qt Creator.
2. Загрузите проект сервера: нажмите Open Project... → найдите и выберите MicroclimateBackend/CMakeLists.txt → сделайте сборку.
3. Загрузите проект панели: аналогично откройте MicroclimatePanel/CMakeLists.txt → сделайте сборку.
4. Запустите сначала сервер (появится консоль с Server started at http://localhost:8080), затем панель (откроется графическое окно).
